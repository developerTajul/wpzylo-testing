== Installation ==

=== From within WordPress ===

1. Visit 'Plugins > Add New'
2. Search for 'Zylo Elementor'
3. Activate Zylo Elementor from your Plugins page.

=== Manually ===

1. Upload the `boomdevs-elementor` folder to the `/wp-content/plugins/` directory
2. Activate the Zylo Elementor plugin through the 'Plugins' menu in WordPress
3. Go to "after activation" below.
